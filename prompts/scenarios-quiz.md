## Description

## 1. Verbes Irréguliers (Irregular Verbs Challenge)

**ID** : `quiz-irregular-verbs`
**Difficulté** : Débutant
**Durée estimée** : 8 minutes
**Vocabulaire clé** : went, saw, took, made, gave, bought, thought, brought

### Étapes

#### Étape 1 - GO
**Instruction pour l'élève** :
> Answer: What is the past tense of GO?

**Prompt pour l'IA** :
```
Start the quiz. Ask: What is the past tense of GO? Wait for answer. Correct answer is WENT.
```

---

#### Étape 2 - SEE
**Instruction pour l'élève** :
> Answer: What is the past tense of SEE?

**Prompt pour l'IA** :
```
Give feedback on previous answer. Ask: What is the past tense of SEE? Correct answer is SAW.
```

---

#### Étape 3 - TAKE
**Instruction pour l'élève** :
> Answer: What is the past tense of TAKE?

**Prompt pour l'IA** :
```
Give feedback. Ask: What is the past tense of TAKE? Correct answer is TOOK.
```

---

#### Étape 4 - BUY (phrase)
**Instruction pour l'élève** :
> Make a sentence using the past tense of BUY.

**Prompt pour l'IA** :
```
Change format. Say: Now make a sentence using the past tense of BUY (bought). Give an example if they struggle.
```

---

#### Étape 5 - THINK (phrase)
**Instruction pour l'élève** :
> Make a sentence using the past tense of THINK.

**Prompt pour l'IA** :
```
Give feedback. Ask for a sentence using the past tense of THINK (thought). Congratulate them on completing the quiz.
```

## 2. Prépositions (Prepositions Master)

**ID** : `quiz-prepositions`
**Difficulté** : Intermédiaire
**Durée estimée** : 8 minutes
**Vocabulaire clé** : in, on, at, time expressions, place expressions

### Étapes

#### Étape 1 - AT (heure)
**Instruction pour l'élève** :
> Fill the blank: I wake up ___ 7 AM every day.

**Prompt pour l'IA** :
```
Start the preposition quiz. Ask them to fill in: I wake up ___ 7 AM every day. Correct answer is AT (specific time).
```

---

#### Étape 2 - IN (mois)
**Instruction pour l'élève** :
> Fill the blank: My birthday is ___ March.

**Prompt pour l'IA** :
```
Give feedback. New question: My birthday is ___ March. Correct answer is IN (month).
```

---

#### Étape 3 - ON (jour)
**Instruction pour l'élève** :
> Fill the blank: The meeting is ___ Monday.

**Prompt pour l'IA** :
```
Give feedback. New question: The meeting is ___ Monday. Correct answer is ON (day of week).
```

---

#### Étape 4 - Multiple
**Instruction pour l'élève** :
> Fill the blank: I live ___ Paris. I work ___ an office ___ the city center.

**Prompt pour l'IA** :
```
Multiple blanks: I live ___ Paris. I work ___ an office ___ the city center. Answers: IN Paris, IN an office, IN the city center.
```

---

#### Étape 5 - Création
**Instruction pour l'élève** :
> Create your own sentence using AT, IN, and ON correctly.

**Prompt pour l'IA** :
```
Final challenge: Create a sentence (or sentences) that uses AT, IN, and ON correctly. Give feedback on their creation.
```

## 3. Vocabulaire (Vocabulary Expansion)

**ID** : `quiz-vocabulary`
**Difficulté** : Intermédiaire
**Durée estimée** : 10 minutes
**Vocabulaire clé** : synonyms, antonyms, adjectives, descriptive words

### Étapes

#### Étape 1 - Synonyme
**Instruction pour l'élève** :
> Give a synonym for HAPPY (a word with similar meaning).

**Prompt pour l'IA** :
```
Start vocabulary quiz. Ask for a synonym of HAPPY. Accept: glad, joyful, pleased, cheerful, delighted.
```

---

#### Étape 2 - Antonyme
**Instruction pour l'élève** :
> Give an antonym for BIG (a word with opposite meaning).

**Prompt pour l'IA** :
```
Good job! Now the opposite: Give an antonym for BIG. Accept: small, tiny, little, miniature.
```

---

#### Étape 3 - Remplacement (positif)
**Instruction pour l'élève** :
> Replace VERY GOOD with a single, stronger word.

**Prompt pour l'IA** :
```
Level up! Replace VERY GOOD with one word. Accept: excellent, fantastic, wonderful, amazing, superb, outstanding.
```

---

#### Étape 4 - Remplacement (négatif)
**Instruction pour l'élève** :
> Replace VERY BAD with a single, stronger word.

**Prompt pour l'IA** :
```
Great! Now replace VERY BAD with one word. Accept: terrible, awful, horrible, dreadful, appalling.
```

---

#### Étape 5 - Application
**Instruction pour l'élève** :
> Use three different words to describe a positive experience you had recently.

**Prompt pour l'IA** :
```
Final challenge: Describe a positive experience using at least three different descriptive words. Encourage variety and give feedback.
```
